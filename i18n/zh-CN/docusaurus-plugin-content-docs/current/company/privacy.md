# 隐私政策

**版本更新日期：2025年12月30日**

欢迎使用 Steedos 低代码平台！我们非常重视您的隐私和数据安全。本政策适用于您通过 Steedos 登录界面进入的所有平台服务。

---

## 1. 我们收集的信息

在您登录及使用 Steedos 平台过程中，我们仅收集实现平台功能所必需的信息：

* **身份认证信息：** 手机号、邮箱、用户名及加密后的密码。
* **组织架构信息：** 当您作为企业成员登录时，我们会记录您所属的租户（Organization）信息及职位权限。
* **开发者元数据：** 包括您在设计器中创建的对象定义、页面配置、逻辑脚本（API/Trigger）等元数据。
* **日志与安全数据：** 登录时间、IP地址、浏览器指纹及操作审计日志（用于保障您的账号安全和故障排查）。

## 2. 数据所有权与隔离

* **数据隔离：** Steedos 采用多租户架构，确保不同企业（租户）之间的数据完全隔离。
* **所有权声明：** 您在平台上通过低代码配置产生业务数据、应用逻辑及代码资产，其**所有权归属于您或您所属的组织**，Steedos 仅作为技术服务平台提供支撑。
* **存储位置：** * **私有云/本地部署：** 若您使用的是 Steedos 开源版或私有化部署版本，所有数据均存储在您自行控制的服务器中。
* **Steedos 云服务：** 若您使用我们的云托管服务，数据将安全地存储在受严格保护的云数据中心。


## 3. 核心功能中的隐私保护

* **权限控制：** 平台提供完善的 **RBAC（基于角色的权限控制）**。除非您显式授权，否则平台维护人员无法查看您的业务数据。
* **API 安全：** 所有通过 Steedos 暴露的接口均受 API Key 或 Session 校验保护。
* **审计追踪：** 平台会自动记录敏感操作（如删除、导出），以便企业管理员进行安全审计。

## 4. 信息共享与披露

我们不会将您的信息出售或共享给任何第三方，除非：

1. **获得您的明确授权：** 例如您主动开启了第三方插件服务（如企业微信同步、钉钉集成）。
2. **法律合规要求：** 响应相关法律法规或司法机关的要求。

## 5. 您的权利

作为 Steedos 用户，您拥有以下权利：

* **访问与导出：** 您可以随时访问并导出您的配置数据。
* **账户管理：** 您可以联系组织管理员修改个人信息或停用账号。
* **注销：** 若您是个人开发者，可在设置中申请注销账号并删除关联数据。
